**TESTCASE** - 1  
**Заголовок**  
Успешное создание задачи с валидными данными

**Предусловие**
- Приложение доступно по адресу `https://avito-tech-internship-psi.vercel.app/`.
- Пользователь может открыть главную страницу приложения.

**Шаги воспроизведения**
1. Открыть главную страницу приложения.
2. Нажать кнопку «Создать задачу».
3. В поле «Название» ввести заголовок.
4. В поле «Описание» ввести произвольный текст.
5. В поле «Проект» открыть выпадающий список и выбрать любое доступное значение.
6. В поле «Приоритет» открыть выпадающий список и выбрать любое доступное значение.
7. В поле «Исполнитель» открыть выпадающий список и выбрать любое доступное значение.
8. Нажать кнопку «Создать».
9. Вернуться на главную страницу (ожидание закрытия модального окна).
10. Проверить, что в списке задач отображается задача с введённым заголовком.

**Ожидаемый результат**
- Форма создания задачи успешно отправляется.
- Модальное окно закрывается.
- На главной странице в списке задач появляется новая задача с указанным заголовком и описанием.
